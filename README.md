# ecommerce-sales-dashboard
E-Commerce Sales Dashboard using SQL Server and Power BI | 100K+ orders analyzed | KPIs, Monthly Trends, Top Categories, Region-wise Sales

## Tools Used
- SQL Server Management Studio
- Power BI Desktop

## Dataset
- Source: Olist Brazilian E-Commerce (Kaggle)
- 100,000+ real orders from 2016-2018

## Project Steps
1. Imported 5 CSV files into SQL Server
2. Wrote SQL queries for KPIs, monthly revenue, top products, region sales
3. Built interactive Power BI dashboard

## Key Insights
- Sao Paulo (SP) contributes highest revenue - 40,500 orders
- Beauty & Health (beleza_saude) is top selling category
- November 2017 was peak revenue month - 1.15M
- Total Revenue: 15.42M across 96K orders
